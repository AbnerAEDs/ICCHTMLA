Como funciona o sistema de controle de versões Git

O git é um sistema de controle de versão distribuido amplamente utilizado para gerenciar projetos de software. Ele permite rastrear alterações em arquivos e coordenar o trabalho de várias pessoas em um projeto. aqui está uma visão geral de como utilizar o git.

Instalação
Para a instalação do git para seu windows, MacOS e linux você pode baixar a versão apropriada para seu sistema operacional no site oficial do Git
Repositório
Após fazer a instalação de umas das verções do git você terá a opção de criar um repositório. O Repositorio é um local aonde se armazena um arquivo e mostra suas alterações do seu projeto, para iniciar um repositório, vá para o diretorio raiz do seu projeto e utilize o comando git init
